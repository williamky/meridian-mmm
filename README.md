# Meridian MMM

Marketing Mix Modeling project using Google's [Meridian](https://github.com/google/meridian) library.

## Setup

```bash
pip install -r requirements.txt
```

## Structure

```
meridian-mmm/
├── data/          # Input data (excluded from git)
├── notebooks/     # Exploratory analysis & model runs
├── src/           # Reusable Python modules
└── outputs/       # Charts, model summaries
```

## Usage

Open `notebooks/01_eda.ipynb` to explore your data, then `notebooks/02_model.ipynb` to fit the MMM.
